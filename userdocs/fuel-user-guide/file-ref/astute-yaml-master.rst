astute.yaml (Fuel Master node)
------------------------------

Fuel Master Node:
**/etc/fuel/astute.yaml**

Fuel uses the *astute.yaml* file to pass configuration attributes
to Puppet.

Usage
+++++

The */etc/fuel/astute.yaml* file is installed
on the Fuel Master node
and must not be deleted.


File Format
+++++++++++
